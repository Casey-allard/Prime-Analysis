# Prime Number Theory & Sieves

This project consolidates prime distributions, phyllotaxis mapping, twin prime conjecture attempts, and massive dataset caches.

## Contents
- `/attempts/`: Sieve scripts and search attempts (attempts 53, 54, 55).
- `/caches/`: Over 50M primes cache in numpy/text formats and Green-Tao bridge models.
- `/results/`: ECDF graphs, phase correlation logs, and residual plots.
- `/diagrams/`: Spiral mappings, phyllotaxis plots, and twin prime candidate candidates.
- `/archives/`: Zipped zero packs and prime lists.
