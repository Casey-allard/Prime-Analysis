# C^2 transport matrix stability across scales

This compares the fitted complex 2x2 transport matrices A for the candidate C^2 quotient
across wheel scales W = 2310, 30030, 510510.

Diagnostics:
- raw matrix entries of A
- eigenvalues and their magnitudes/angles
- pairwise matrix distance
- pairwise approximately-conjugated distance
- top torus-mode loadings contributing to z1 and z2

Interpretation:
- stable eigenvalues / similar conjugacy class => evidence of a reusable global transport law
- drifting matrices / unstable spectra => scale-dependent quotient rather than universal one
