
# QIMC Observer Decoherence Simulation

This simulation demonstrates Quantum Information Merge Conflicts (QIMC) through the decoherence trajectories of multiple observers on a single qubit system. Each observer experiences decoherence in a different basis (Z, X, Y).

## Files

- `qimc_simulations.py`: Run this to simulate and generate plots.
- `requirements.txt`: Install dependencies using `pip install -r requirements.txt`.
- `observer_decoherence_plot.png`: Output plot comparing observer state divergence.

## How to Run

```bash
pip install -r requirements.txt
python qimc_simulations.py
```
