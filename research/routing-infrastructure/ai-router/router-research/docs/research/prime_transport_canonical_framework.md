It records the primary mathematical object of the project and separates that
exact layer from predictive compression and from downstream approximations.

The source-of-truth framework is the discrete recursive admissibility system,
not any downstream quotient model.

## Exact Definitions

### 1. Wheel and Orbit

For wheel depth `r`, let

- `W_r = 2 * 3 * 5 * 7 * Π q_m`
- `L_r = W_r / 6`
- `j in Z / L_r Z`
- `n_r(j) = 5 + 6j (mod W_r)`

The odometer dynamics is

- `j -> j + 1`

All admissibility structure is built on this single cyclic orbit.

### 2. Exact Admissibility Bit

For offsets `a in A`, define

- `w_r(j) = 1`
  iff `gcd(n_r(j) + a, W_r) = 1` for every `a in A`

The cyclic word `w_r` is the exact symbolic dynamics of admissibility along the
transport orbit.

### 3. Phase-Fiber-Scale Factorization

At finite depth, the exact state is

- `j -> (b(j), phi(j), r)`

with

- `b(j)` the base phase
- `phi(j)` the refinement fiber phase
- `r` the wheel depth

This is the exact finite-depth state description. The phase-fiber layer is not
an approximation.

## Theorem-Shaped Exact Objects

### 1. Universal Local-Stencil Factorization

For each odd prime layer, admissibility is determined locally by allowed and
forbidden residues modulo that prime. Global admissibility is obtained by CRT
gluing of these local stencils.

So admissibility is not arbitrary global filtering; it is a product of local
stencil constraints assembled by CRT.

### 2. Recursive Affine Lift Law

For a lift `W -> pW`, the deeper admissibility system is generated from:

- a local stencil modulo `p`
- an affine translation inherited from the lower wheel

This is the recursive generation law for new wheel layers.

### 3. Fiber Evolution Theorem

After affine normalization, the fiber admissibility set evolves by direct CRT
product:

- `U^*_{A,r} = CRT(U^*_{q_1} x ... x U^*_{q_r})`

This expresses exact fiber evolution as a multiplicative CRT construction, not
as an empirical fit.

### 4. Symbolic Dynamics Interpretation

The admissibility word `w_r` is a symbolic coding of the odometer orbit. The
transport system may therefore be studied as a symbolic dynamical system built
from recursively lifted local constraints.

### 5. Finite-Horizon Spin

For horizon `H >= 1`, define

- `spin_H^W(j) = (w_W(j), w_W(j+1), ..., w_W(j+H-1))`

with cyclic indexing modulo `L_W`.

The compressed predictive state is

- `sigma_H^W(j) = (b(j), spin_H^W(j))`

This is the correct predictive compression object at the exact-system layer.

### 6. Delayed Visibility

For a lift `W_parent -> W_child`, new refinement layers may activate in two
stages:

- **internal split**:
  futures over child lifts above the same parent position begin to diverge
- **visible split**:
  the count of compressed predictive states `(b, spin_H)` increases

Internal split can happen strictly before visible split. This is the
delayed-visibility phenomenon.
