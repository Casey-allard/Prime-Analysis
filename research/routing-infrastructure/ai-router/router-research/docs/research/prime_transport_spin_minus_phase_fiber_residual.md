# Prime Transport Spin Minus Phase-Fiber Residual

## Question

What extra predictive information does finite-horizon spin carry beyond the
static exact chart `j -> (b, phi, r)`?

Since `j -> (b, phi, r)` is already an exact coordinatization of the transport
orbit, this is not a question about raw information completeness. It is a
question about **compressive predictive explanation**.

The exact residual question is therefore:

- can the predictive distinctions carried by `(b, spin_H)` be represented by a
  small exact dynamical object on the orbit, rather than rebuilding the full
  future word from scratch?

## Setup

The bounded experiment used the two already-constructed tightly density-matched
families:

- `2310 -> 30030`
- `30030 -> 510510`

For each row:

1. Take the known first visible split horizon `H_visible_first`.
2. Form the pre-threshold predictive target `(b, spin_{H_visible_first-1})` on
   the parent orbit.
3. Restrict attention to the parent positions that belong to the first visible
   splitting support.
4. Ask whether a small exact dynamical label on those positions can reproduce a
   substantial portion of the predictive partition.

The candidate family was intentionally small:

- recent admissibility suffix of length `3`
- recent admissibility suffix of length `5`
- local transition triple `(w_{t-1}, w_t, w_{t+1})`
- next return gap
- previous/next return-gap pair

Each candidate was tested as a small exact dynamical label paired with base
phase `b`.

## Result

The strongest candidate was:

- `prev_next_gap_pair`

Its mean performance across the bounded matched-family table was:

- mean purity gain over `b` alone: `0.2484044128966601`
- mean candidate exact purity fraction: `0.46856477589264134`

The next best candidate was:

- `next_return_gap`

with:

- mean purity gain over `b` alone: `0.17152481542797807`
- mean candidate exact purity fraction: `0.39168517842395933`

The short suffix and local transition candidates were much weaker:

- `local_transition_3`: mean exact purity `0.25699186120830214`
- `recent_suffix_3`: mean exact purity `0.2475072250796963`
- `recent_suffix_5`: mean exact purity `0.2475072250796963`

The row-level pattern matters:

- on the smaller `2310 -> 30030` family, `prev_next_gap_pair` often raises
  exact purity into the `0.50 - 0.63` range
- on the deeper `30030 -> 510510` family, the same candidate drops to roughly
  `0.11 - 0.26`

So the current bounded evidence does **not** support the claim that a single
small return-memory object already replaces full spin.

## Interpretation

The best current reading is:

- the residual beyond static phase-fiber-scale coordinates is not arbitrary
- return-memory information is more relevant than short local suffix memory
- a previous/next return-gap pair is the strongest small exact dynamical object
  tested so far
- but it captures only part of the predictive partition generated by spin

This points away from a purely local bit-suffix explanation and toward a
slightly richer return-grammar object.

The conservative conclusion is:

- the residual looks **partially compressible** into a small exact dynamical
  object
- but full spin still appears necessary on the current evidence, especially on
  the deeper lift family

## Status

This should be treated as a bounded exact-layer refinement of the current
phase-fiber-versus-spin picture:

- phase-fiber-scale coordinates are exact and real
- spin remains the stronger predictive compression
- the best current residual candidate class is return-memory rather than short
  local suffix memory
- no small exact residual object has yet earned replacement status for full
  spin
