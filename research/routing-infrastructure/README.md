# Routing & Infrastructure

This directory houses the routing algorithms, infrastructure engines, and networking/observability specifications.

## Projects Included

1. **[r4-prime-router-wasm](r4-prime-router-wasm/)**: A high-dimensional continuous R4 tangent space router rebased under the official UOR Framework and Prism Model. It features a compiled Rust backend, WASM client-side fallback, and a 3D Hopf fiber visualizer.
2. **[ai-router](ai-router/)**: Early Python prototypes, diagnostics tools, configurations, and results for prime transport semantic packet routing.
