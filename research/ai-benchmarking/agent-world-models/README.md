# Agent World Models & Runtime Specs

This project contains configurations and installers for persistent multi-agent coordination.

## Contents
- `/workerrepo/`: Agent state snapshots, logs, and prompt comparisons.
- `Copilot agent.agent`: Agent definition specifications.
- `FOUR_SUBAGENT_PAPER.md`: Multi-agent synthesis paper drafts.
- `Technical Architecture Review...pdf`: Review of the Universal Agent Runtime (UAR) specification.
- `agent_installer*.sh`: Installation binaries for ARM64 and x86_64 architectures.
