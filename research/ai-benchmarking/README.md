# AI & Benchmarking

This directory contains multi-agent evaluation suites, benchmarking specifications, persistent identity models, and deployment configurations.

## Projects Included

1. **[mudbench-core](mudbench-core/)**: Multi-agent text-world environment for benchmarking persistent coding agents.
2. **[agent-world-models](agent-world-models/)**: Universal Agent Runtime (UAR) specifications, Copilot agent configurations, and deployment installer scripts.
3. **[mudbench-proposals](mudbench-proposals/)**: Commercial proposals and design specifications for SkyNetwork integration.
